Reverse Engineering with Radare2
December 25, 2016

Level 1

You can view the binary here.
Here, we need to do a little more debugging to find the password. By disassembling the binary, we can see that it contains the following lines in the main method.

| 0x0804841f c704244c8504. mov dword [esp], 0x804854c ; [0x804854c:4]=0x49006425 ; "%d"
| 0x08048426 e8e1feffff call sym.imp.scanf ; int scanf(const char *format);
| 0x0804842b 817dfc9a1400. cmp dword [ebp - local_4h], 0x149a ; [0x149a:4]=0x2ec0804
| ,=< 0x08048432 740e je 0x8048442

We can patch it in much the same way as level0. Simply jump to the address 0x08048432 and replace the je with a jmp.
