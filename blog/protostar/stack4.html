Exploit Exercises(protostar) writeup
December 22, 2016

Level 4(Stack3)

void win() {
    printf("code flow successfully changed\n")
}
int main(int argc, char **argv) {
    volatile int (*fp)();
    char buffer[64];
    fp = 0;
    gets(buffer);
    if(fp) {
        printf("calling function pointer, jumping to 0x%08x\n");
        fp();
        }
    }
}

Here, we need to overwrite the variable fp with the address of win(). We can debug the program with GDB to find the address. The exact command is "p &win". Then, once we have found the address, we simply fill up the buffer with A's, and overwrite fp with the address we have. Check it out here.
