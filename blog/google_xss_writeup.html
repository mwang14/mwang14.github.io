Google XSS game writeup
September 5, 2016

One of my friends has expressed interest in learning web application security, so I recommended Google's XSS game. I tried this back in February and got stuck on the second level. This time, I was able to get through it much faster.

Level 1 was just inputting a script tag with an alert into the search field.

Level 2 filtered the script tag, so I inputted an image tag that popped up an alert if there was an error.

Level 3 was exploiting an unsanitized image tag. After escaping from the original string, I was able to input an onerror tag that would pop up an alert.

Level 4 was another unsanitized input. This time, I had to escape it with a single quote, but it filtered out semicolons. I put in the encoded version of a semicolon and was able to get an alert tag to pop up.

Level 5 was(again) an unsanitized input. There was an unsafe href tag, and I was able to put an alert statement into the URL.

Level 6 (finally) had some sanitization of their inputs. However, it didn't filter out capital letters. After putting in a link with an alert statement, an alert statement popped up.




I thought this was a good challenge, especially for beginners. Definitely a good resource to start learning XSS.
